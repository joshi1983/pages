Rem Circles 256 colors
Screen _NewImage(640, 480, 256)
Do
For s = 1000 To 2000 Step 250
Cls
For y = 45 To 700 Step 15
c = c + 1
Circle (y, 200), 40, c
Locate 20, 36
Print "color"; c
_Delay 0.1
Sound s, 0.5
Next y
Next s
Loop While c < 220
