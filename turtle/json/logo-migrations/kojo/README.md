colors.json contains data converted/adapted from Kojo's source code.
colors.json was adapted from src/main/scala/net/kogics/kojo/doodle/CommonColors.scala.

WebLogo has a collection of named colours but Kojo's names and definitions for its names might be different.
This data is used to help the translator maintain the colours as defined in Kojo while translating to WebLogo.  
Where differences exist, a hex representation is used instead of a colour name.
