We can draw a square like this:

forward 100
right 90

forward 100
right 90

forward 100
right 90

forward 100
right 90

That duplicates a lot of code, though. It is long, repetitive, and that would get worse if you wanted to draw a shape with more sides. There is a better way to repeatedly run instructions and below is an example.

repeat 4 [
  forward 100
  right 90
]

You wrap the instructions you want repeated in square brackets [] and say how many times you want to repeat them.

The repeat command enables you to draw much more complex drawings without copy/pasting lots of code. It helps your code stay understandable for you and anyone else reading your code.

Here is a 360-sided regular polygon. This has so many sides that it looks like a circle.

repeat 360 [
  forward 2
  right 1
]

The following loop draws a 5-pointed star.

repeat 5 [
  forward 100
  right 72
  forward 100
  left 144
]

You may want to change what your repeated instructions do based on how many times the loop has run. The repcount command works in a repeat loop to give you the number of repetitions that already completed.

The following code reads the repetition count to print 1 2 3 4 5.

repeat 5 [
  print repcount
]

Increasing how far you move forward in relation to turning can lead to a spiral pattern. This idea is applied below to make a spiral pattern using repcount.

repeat 100 [
  forward repcount
  right 20
]

Example Printing Random Even/Odd Messages

The following example uses repeat to print 10 random numbers along with saying if each is even or odd.

repeat 10 [
  make "randomNumber random 100
  ifelse (remainder :randomNumber 2) = 0 [
    print [:randomNumber "is "even]
  ] [
    print [:randomNumber "is "odd]
  ]
]
