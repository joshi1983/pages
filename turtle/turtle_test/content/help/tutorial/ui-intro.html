Why learn Logo?

  • Logo is fun, rewarding, and helps you be creative
  • You can make drawings more detailed and complex than ever possible on paper
  • Logo is a great tool for applying math skills
  • Logo is easy to learn. WebLogo helps beginners get started by giving some of the fastest and most thorough feedback as code is typed. Code analysis feedback as you type is a feature not available in most programming environments but it is very helpful to anyone just starting to program.

WebLogo is easy to learn. WebLogo is focused on creating drawings, animations, and 3D models. That focus helps WebLogo be one of the best Logo programming environments for graphics.

Upon opening WebLogo, you'll see two sections.

  • Graphics Screen - a place to view the current drawing
  • Commander - a place to enter single-line commands

We will run commands in the text input at the bottom of the Commander. Click the "Run" button or press the ENTER key to run your commands.

The Commander also has an area above the command input box to show a history of previously run commands.
