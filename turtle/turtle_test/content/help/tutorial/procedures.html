Now that you've used several commands, variables, and loops, you may want to create your own command out of them.

You can draw a square like this:

repeat 4 [
  forward 100
  right 90
]

You can also make that drawing available the same way you call a command by defining a procedure like the following.

to square
  repeat 4 [
    forward 100
    right 90
  ]
end

square ; call it like a command
