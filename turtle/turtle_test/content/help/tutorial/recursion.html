The forward command draws a solid line if it draws one at all. No command draws a dashed line but you can make your own that does. "forwardDashed" sounds like a good name for our new command.

How would you draw a dashed line?

The forward, and jumpForward commands let you intermittently draw short solid lines like this:

forward 5 jumpForward 5 forward 5 jumpForward 5

You could reduce some of the code duplication with a loop like this:

repeat 2 [   forward 5
  jumpForward 5
]

You might have other drawings that currently use the forward command and you want to replace those solid lines with dashed lines.

If you call forward in more than one place, you may be tempted to paste the above code into multiple places.

forward 100
right 45
repeat 4 [
  forward 100
  left 90
]

The following code could make a similar drawing using dashed lines if we teach WebLogo how to do "forwardDashed".

forwardDashed 100
right 45
repeat 4 [
  forwardDashed 100
  left 90
]

The following is how we can define the forwardDashed command:

to forwardDashed :len
  if :len > 0 [
    forward 5
    jumpForward 5
    forwardDashed :len - 10
  ]
end

forwardDashed 100
right 45
repeat 4 [
  forwardDashed 100
  left 90
]
