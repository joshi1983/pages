A variable is a symbol with an associated value. The value can vary or change but the name of a given variable is always the same.

You can assign a value to a global variable using the make command.

The following example makes the variable named x have the value 3 and then prints it.

make "x 3
; Makes x equal to 3.

print :x
; prints the value of x which is 3

Variables help make your code more flexible and less repetitive. Consider the following code that draws a triangle:

forward 100
right 120
forward 100
right 120
forward 100
right 120

If you wanted the square to have sides of length 150 instead of 100, you'd have to update 3 lines of code to this:

forward 150
right 120
forward 150
right 120
forward 150
right 120

After using a variable to represent the length of each side, you can change the size of every side by modifying only 1 line.

make "sideLength 150
forward :sideLength
right 120
forward :sideLength
right 120
forward :sideLength
right 120

A good variable name also helps you describe what you're representing to make your code more understandable to the next person who reads it.

It also must follow these rules:

  • A variable name must not contain any spaces.
  • A variable name can start with underscore(_) or a letter.
  • Any character after the first can be a period(.), underscore(_), a digit, or a letter.
  • A variable name can end with a question mark(?). The ? can not appear anywhere except at the end.
