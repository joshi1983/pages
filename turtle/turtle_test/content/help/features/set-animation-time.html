Clicking Set -> Animation Time opens a dialog for setting the time that the animation.time command returns. The selected time applies to code run from the Commander or when you run your code from the Code Editor.

Live Redraw

Check Live Redraw to see the effect of a newly selected time by having WebLogo clear, rerun, and redraw the drawing in the dialog's background as you drag the selected time. The Live Redraw checkbox is checked automatically as the dialog opens whenever the current program makes use of the animation.time command.

Uncheck Live Redraw if you don't like having it redraw in the background

Selecting a time

You can select a time in a couple different ways.

  • You can click or drag the range selector.
  • Click on the seconds and you can type a number of seconds or use the spinner's up/down buttons.
