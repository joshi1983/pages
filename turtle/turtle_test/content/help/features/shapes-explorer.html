The Shapes Explorer is accessed by clicking Drawing -> Shapes in the main menu.

The Shapes Explorer lets you see the sequence of shapes used in your current drawing in an interactive spreadsheet-like format. The graphics screen shows the drawing as a single image which is generally the most useful. Shapes Explorer could help with:

  • troubleshooting. By showing all details about every shape, you can learn more about how mistakes in code lead to unintended changes in the shapes of the drawing.
  • optimizing your code to produce a drawing with fewer or simpler shapes. How many shapes can be a useful metric to optimize for while retaining an intended general appearance.
