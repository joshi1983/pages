Pen contact is whether the pen is up or down. Similar to a pen and paper, the pen must be pressed down when it moves or no lines are drawn.

The pen can be pressed down using the pendown command. The pen can be lifted up using the penup command. You can also use the Set -> Lift Pen Up or Set -> Put Pen Down menu item.

You can see if the pen is down by reading either "Up" or "Down" in the status feature. In a Logo program, you can also use the penDown? command.
