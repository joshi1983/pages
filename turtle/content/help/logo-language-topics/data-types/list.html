A list contains many values. A list is usually indicated with square brackets and each element is separated by any whitespace.

An example of a list is: [1 2 3].

A list can be printed using the print command.

Lists can represent many different values in a WebLogo program such as colors, semitransparent colors. Lists could represent vectors and matrixes for use with linear algebra calculations. Lists could be used to contain many values any other time you need a contain.

A few commands often used with lists include:

  • count for counting length of a list
  • item for getting the value of an element at a specific index in a list
  • setItem for setting the value of an occupied element in a list
  • queue for adding a new element to the end of a list
