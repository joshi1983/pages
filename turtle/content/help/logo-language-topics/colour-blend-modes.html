Color blend modes determine the final color after drawing a foreground color over a background color.

The color blend mode can be set using commands like setPenBlendMode and setFillBlendMode. You can read the current blend modes with commands like penBlendMode and fillBlendMode. Essentially any drawing commands will be affected by the current blend modes. A few such drawing commands include forward, image, imageAlpha, isoTriangle, rect, regularPolygon, stripes but there are many more.
