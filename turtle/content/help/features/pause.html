Pause is a button on the right side in the Commander. It is one of the buttons in the lower right area of the initial screen layout.

Clicking the button pauses or unpauses execution of your program. The button is enabled only when a program is started but not halted. You'll see the button's text is gray instead of black when it can't affect anything. The text toggles between "Pause" and "Unpause" to describe what happens when the button is clicked again.

Practice Exercise

Follow these steps to see the pause feature work.

  1. Set the program execution to very slow. Click Debugging -> Very Slow. The initial execution speed is Maximum Speed.
  2. Copy the following code into the Commander input field: repeat 1000 [ print repcount ]
  3. Click the Run button on the right side of Commander input to execute the code. You should see it print 1 2 3... at roughly 1 number per second
  4. Click the Pause button. Notice it stops printing.
  5. Click Unpause. Notice it starts printing numbers wherever it left off. If it already printed 1 2 3, it will print 4 5 6...
