Status shows the state of the turtle including heading, pen width, pen color, fill color, screen color...

How to change the Status

Use the Set menu or various commands if you want to change various properties shown in the Status feature.

  • Pen Contact can be changed with the pendown and penup commands.
  • Pen width can be changed with the setpensize command or Set -> Pen Size from the menu.
  • Turtle position can be changed with the setpos, setxy, setxyz commands.
  • Turtle visibility can be changed with the hideturtle, showturtle commands.
  • Pen Color can be changed with the setpencolor command.
  • Shape Fill Color can be changed with the setfillcolor command.
  • Screen Color can be changed with the setscreencolor command.
