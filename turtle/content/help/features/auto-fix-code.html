The Code Editor's Edit -> Auto-fix Code menu item performs various fixes on code in the editor.

The auto-fix feature can help anyone migrate code from other Logo programming tools and fix frequent minor mistakes and typos.

The following are a few specific fixes performed by the Auto-fix Code feature.

  • Converting code from other Logo programming tools to WebLogo. Below are some examples:
    • if [3 < 5] [] is converted to if 3 < 5 [].
    • setPenSize [5 5] is converted to setPenSize 5.
    • pc 5 is converted to setPenColor 5.
  • Fixes to common mistakes for people who are new to Logo programming. For example [1,2,3] gets converted to [1 2 3] since WebLogo does not expect elements in a list to be separated by commas.
  • Fixes some cases where quotes are missing. Here are some examples:
    • setPenColor red is converted to setPenColor "red
    • make x 5 is converted to make "x 5
