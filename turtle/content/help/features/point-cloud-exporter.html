The Point Cloud Exporter is available for some drawings after clicking Drawing -> Download.

The Point Cloud Exporter lets you download files that represent various points from the drawing.

Limitations

Not all drawings can be exported to a point cloud. The only requirement is that your drawing include at least 1 shape with an exportable point with z-coordinate other than 0. Generally any model that you intend to be 3D can be exported to a point cloud.
