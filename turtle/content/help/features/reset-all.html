The Reset All feature is triggered from the Code Editor's menu at ! -> Reset All.

Reset All does the following:

  • The drawing is cleared. All shapes are removed. The screen color goes transparent.
  • The turtle is reset. This includes the turtle to position [0 0 0], gets a heading of 0, pen color of black, fill color of transparent.
  • All global variables are removed from memory.
  • The running program is halted, if it is running. This is similar to clicking the pause button but permanent. You can't unpause a halted program.
