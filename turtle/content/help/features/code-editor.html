The Code Editor is where you edit code for large Logo programs.

The Code Editor can be opened in a few ways.

  • Click File -> Edit
  • Click the Edall button in the Commander.

When multiple lines of code are selected, the following features can help you edit code more quickly.

  • Press the semicolon (;) to comment all selected lines at once.
  • Hold down the CTRL key while hitting semicolon (;) to uncomment every selected line
  • Hit the TAB key to indent all selected lines
  • Hold down SHIFT while hitting TAB to unindent all selected lines
