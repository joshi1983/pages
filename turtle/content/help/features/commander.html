The commander shows printed messages and lets you enter single-line Logo scripts.

If you want to implement procedures or use more lines of code, open the code editor. The code editor can be opened through the menu at File -> Edit or by clicking the "Edall" button on the right side of the commander.

The printed messages may include:

  • problems with code you tried to run
  • messages from the print command if you called it from your code.
  • single-line commands that you tried to run

Clear Text

The clear text button is on the upper right corner of the commander.

Clicking the button clears all messages in the commander.

Hiding/Showing Commander

The commander can be hidden or shown by clicking Debugging -> Show Commander

Resizing Commander

The height of the Commander can be controlled by dragging the Commander title up or down.
