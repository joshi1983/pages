The Quality Report can be opened by clicking Testing -> Quality Report in the Code Editor's menu.

The quality report helps you find a variety of bugs in your code.

The quality report lists:

  • It shows all the messages you can see when you click to run your program.
  • It also runs your code many different ways to look for runtime errors. If you're defining an animation, it'll run with different animation.time values. If you use randomized commands, it'll run with different initial pseudorandomization seeds.
