The image exporter is available for all drawings after clicking Drawing -> Download.

The image exporter lets you download images of the drawing.

The following explains the image exporter menu.

File Formats

  • JPG is a very popular format that compresses well but does not support transparency.
  • PNG is a very popular format that supports transparency.
  • SVG is a vector-graphics format that is great for icons and logos. SVG doesn't store data by pixels but instead stores drawing information similar to what your WebLogo code expresses. The main advantage of SVG over other formats is that it is that you'll never see pixelation in it unless you embed another rasterized image using the image command.
  • WEBP is a newer standard format mostly used in websites.

Resolution

The resolution menu lets you select the exported image dimensions. You can select from a predefined list of popular standard dimensions or select a custom resolution by individually setting width and height.

Zoom

You can use the mouse wheel, if you have one to zoom. You can also use the zoom and nudge menu items as described below.

  • Zoom In lets you zoom in in 10% increments.
  • Nudge In lets you zoom in in 1% increments.
  • Zoom Out lets you zoom out in 10% increments.
  • Nudge Out lets you zoom out in 1% increments.
