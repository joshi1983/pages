Assets can be accessed from the main menu by clicking File -> Assets.

Assets are information resources that are saved in your web browser. They can be images or any other type of file data.

You can add an image from .png, .jpg, or .svg format and draw it using the image command by referencing it with a URL such as local://myimage.png assuming you named your asset myimage.png.

If you view an existing image asset, you can add the code for drawing it by clicking the "Add to Code" button.
