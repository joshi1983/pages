The debugger can be accessed by clicking Debugging -> Debugger in the main menu.

The Debugger lets you look deeply at the state of your running Logo program to investigate problems.

Global Variables

The global variables section lists all global variables and their current values.

Global variables are all variables that are not local. Global variables are often created while calling the make command but can also be made by for-loops that are not within procedures.

Call Stack

Call stack lists all procedure calls that are currently running, if there are any. Call stacks also shows all parameters and local variables with corresponding values.

Repcount Stack

Repcount Stack shows the state of all repeat commands currently executing including their maximum and current values. Unlike a for command, the repeat command has its counter value accessible through the repcount command instead of a variable so the state of a repeat loop does not show in any other section of the debugger.
