The repeat command is very helpful kind of loop but it isn't the only option.

For-Loops

The repcount command used with repeat returns a number starting with 1 and increasing by 1 but you might want to start with something other than 1 and increase by something other than 1. The for is great for that.

The following example prints 0 0.5 1 1.5 2 2.5 3

for ["x 0 3 0.5] [
  print :x
]
  • "x is a variable name. It is given a different value each time the instruction list runs.
  • 0 is the value given to variable x in the first run of your repeated instructions.
  • 3 is the stop value. The loop ends when x becomes equal to or beyond the stop value.
  • 0.5 is the step value. The step is added to x each time the repeated instructions run.

While-Loops

Instead of counting how many repetitions you want or adjusting a variable, you might have a more general condition in mind.

The following example prints random numbers while the random number is not 8. This can't be done as well with for or repeat loop because the number of repetitions is not the point of our loop condition.

make "randomNumber 0
while :randomNumber <> 8 [
  make "randomNumber random 10
  print :randomNumber
]

Until-Loops

An until is very similar to while except that the condition means the opposite. Until loops repeat until the condition is true instead of looping while the condition is true.

The example below behaves the same as the while-loop above except using an until command.

make "randomNumber 0
until :randomNumber = 8 [
  make "randomNumber random 10
  print :randomNumber
]

Do-While Loop

A do-while-loop is similar to a while-loop except that the condition is checked after each run of the repeated instructions.

Since the condition is checked only after a random number is assigned to randomNumber, the make "x 0 can be removed as shown below.

do.while [
  make "randomNumber random 10
  print :randomNumber
] :randomNumber <> 8

Forever Loops

The forever loop is available if you want the loop to never stop.

The following example will endlessly print random numbers from 0 to 99(inclusive).

forever [
  print random 100
]
