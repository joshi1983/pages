The simplest way to color a filled shape like a circle is to set fill color using the setFillColor command.

setFillColor "red
circle 100
; fills a circle with a solid red color

You might want to fill a shape with a blend of colors. A gradient is an area that transitions between more than 1 color.

Defining a gradient requires specifying color stops.

  • You must indicate a ratio from 0 to 1 for each color stop. This ratio indicates where the color appears in the gradient.
  • Color stops also require the color. The following indicate the color to be red: "red, [255 0 0], or color index.

The following defines a variable colorStops for use in a gradient:

make "colorStops plistCreate ; create a property list. setProperty "colorStops 0 "red setProperty "colorStops 1 "black ; the gradient blends between red and black.

The commands for creating a gradient include:

  • createLinearGradient - for blending between colors in a single straight direction. The perpendicular directions with constant color are also straight.

  • createRadialGradient - for blending between colors in a radiating direction. Perpendicular directions with constant color are circular.

make "colorStops plistCreate setProperty "colorStops 0 "red setProperty "colorStops 1 "black ; the gradient blends between red and black. setFillGradient createLinearGradient [0 0] [100 0] :colorStops "pad circle 200 ; draws a circle filled with a gradient that blends ; between red and black ; Point [0 0] is red. ; point [100 0] is black. ; Point [50 0] is an even mix between red and black.
