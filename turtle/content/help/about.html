WebLogo was created for educators, aspiring learners, and artistic creators.

WebLogo is a web application inspired by FMS Logo. WebLogo was made from scratch but shares a lot of features and commands to help anyone familiar with different Logo programming environments. Key differences are explained here.

WebLogo was started by Josh Greig in 2022. Josh loves programming and Logo was the language that first sparked that passion. WebLogo was created to share that language and passion.
